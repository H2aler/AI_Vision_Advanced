🎯 기본 기능

📹 카메라 시작
웹캠을 활성화하여 실시간 분석을 시작합니다. 모바일에서는 후면 카메라가 우선 사용됩니다.
⏹️ 카메라 중지
웹캠을 비활성화합니다.
📁 이미지 업로드
컴퓨터에서 이미지 파일을 선택하여 분석합니다. 지원 형식: JPEG, PNG, GIF, WebP
🎞️ 동영상 업로드
컴퓨터에서 비디오 파일을 선택하여 분석합니다. 지원 형식: MP4, WebM

🤖 AI 분석 기능

📐 3D 객체 감지
이미지/영상에서 객체를 감지하고 3D 공간에서의 거리와 위치를 분석합니다.
🎯 실시간 추적
웹캠이나 비디오에서 객체를 실시간으로 추적하고 이동 경로를 시각화합니다.
🧍 포즈 추정
사람의 포즈를 감지하고 33개 키포인트를 분석합니다.
🔄 실시간 포즈
실시간으로 포즈를 추정하고 분석 결과를 표시합니다.
🔄 이미지 세그멘테이션
인체와 배경을 분리하고 세그멘테이션 분석을 수행합니다.
😊 얼굴 랜드마크
468개 얼굴 키포인트를 감지하고 얼굴 특징을 분석합니다.
🎨 AI 이미지 생성
이미지 분석을 바탕으로 AI가 스타일을 변환한 새로운 이미지를 생성합니다.
🔬 다중 모달 분석
8개 AI 모델을 동시에 실행하여 종합적인 분석 결과를 제공합니다.
🌍 천체 감지
지구, 화성, 달, 행성 등의 천체를 자동으로 감지하고 분석합니다. 색상과 형태 기반 분석
🌍 고급 자연 객체 감지
지구, 구름, 대륙, 바다, 폭풍, 우주 등 모든 자연 현상을 감지합니다. 세계 모든 객체 인식
👥 확장된 객체 감지
사람(성인/청소년/아이), 동물(강아지/고양이), 차량(자동차/소방차/경찰차/비행기), 물건 등 모든 객체를 감지합니다.
🧠 행동 예측 AI
사람과 동물의 표정, 손짓, 행동을 분석하여 미래 행동을 예측합니다. 감정 + 제스처 + 행동 패턴 분석
🚨 범죄 예방 AI
슈퍼마켓, 매장, 백화점, 식당에서 의심스러운 행동을 감지하여 범죄를 예방합니다. 표정 + 손짓 + 행동 패턴 + 물건 상호작용 분석

📄 OCR 기능

텍스트 인식
기본 텍스트 인식 기능입니다.
고급 OCR
다국어 지원 및 전처리가 적용된 고급 텍스트 인식입니다.
INVOICE 전용
송장 문서에 특화된 텍스트 인식으로, 총액, 날짜, 인보이스 번호를 자동 추출합니다.
서명/도장/로고
서명, 도장, 회사 로고 등 특수 요소를 인식합니다.

🎨 화질 향상

🔍 이미지 화질 향상
단일 이미지의 화질을 2배 업스케일링하고 노이즈를 제거합니다.
🎬 영상 화질 향상
실시간으로 비디오 프레임의 화질을 개선합니다.

🎓 사용자 학습 (KNN)

샘플 추가

KNN 예측

초기화
